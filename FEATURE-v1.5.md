# ✨ 新增功能：空状态快速添加账号

## 功能说明

当网站暂无保存账号时，面板不再显示单调的"暂无账号"，而是展示**快速添加表单**，用户可直接在面板内添加账号，无需跳转到管理界面。

## UI 效果

```
┌──────────────────────────────┐
│ ➕ 快速添加账号              │  ← 新：快速添加标题
├──────────────────────────────┤
│ 用户名                       │  ← 标签
│ [qingyao         ]           │  ← 预填（从页面读取）
│                              │
│ 密码                         │  ← 标签
│ [••••••••        ]           │  ← 预填（从页面读取）
│                              │
│ 标签（可选）                 │  ← 标签
│ [如：测试账号    ]           │  ← 可选字段
│                              │
│     [💾 保存账号]             │  ← 保存按钮
└──────────────────────────────┘
```

## 工作流程

### 1️⃣ 用户访问没有保存账号的网站

```
用户访问登录页
    ↓
聚焦用户名或密码框
    ↓
面板弹出
```

### 2️⃣ 面板展示快速添加表单

```
if 该网站有保存账号:
    显示账号列表 + "管理密码..."
else:
    显示快速添加表单 ✨
```

### 3️⃣ 用户快速填充或手动输入

```
用户名框：自动预填页面上已输入的值
密码框：自动预填页面上已输入的值
标签框：手动输入（可选，如"测试"、"生产"）
    ↓
点击"💾 保存账号"
    ↓
保存成功 → 面板刷新 → 显示刚添加的账号
```

### 4️⃣ 直接点击账号登录

```
面板中显示刚添加的账号
    ↓
点击账号
    ↓
自动填充用户名和密码
    ↓
点击登录
```

## 技术细节

### 自动预填逻辑

```javascript
// 读取页面上已输入的值
const usernameInput = findUsernameInput(inputEl);
const passwordInput = findPasswordInput(inputEl);
const pageUsername = usernameInput?.value || '';
const pagePassword = passwordInput?.value || '';

// 预填表单
usernameInputEl.value = pageUsername;
passwordInputEl.value = pagePassword;
```

**场景 1**：用户已输入用户名，点击密码框打开面板
- 用户名自动预填 ✓
- 密码框为空（用户还没输）
- 用户输入密码后保存

**场景 2**：用户已输入用户名和密码，点击密码框打开面板
- 用户名自动预填 ✓
- 密码自动预填 ✓
- 直接点击保存 ✨

### 保存逻辑

```javascript
chrome.runtime.sendMessage({
  action: 'savePassword',
  hostname,                    // 当前网站
  username,                    // 用户名
  password: pwd,              // 密码
  label                       // 标签（可选）
}, (response) => {
  if (response.success) {
    // 关闭空状态面板
    closePasswordModal();
    // 重新打开面板以显示刚添加的账号
    showPasswordModal(inputEl);
  }
});
```

### 防止干扰

- ✓ 表单内的 `<input>` 设置 `_pwFillerListened = true`，防止触发 `detectLoginForms`
- ✓ 密码框用 `type="text"` 显示（避免再次触发密码检测）
- ✓ `passwordModal.addEventListener('click', e => e.stopPropagation())` 防止冒泡关闭
