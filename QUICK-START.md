# 快速开始指南

## 5 分钟内快速启动

### 步骤 1: 生成图标 ⚙️

选择以下任一方式：

#### 🐍 Python 方式（推荐，最简单）
```bash
# macOS/Linux
python3 generate-icons.py

# Windows
python generate-icons.py
```

需要安装 Pillow：
```bash
pip install Pillow
```

#### 📦 Node.js 方式
```bash
npm install
npm run generate-icons
```

#### 🖼️ 手动方式
在 `icons/` 文件夹中放入三个 PNG 图标文件：
- `icon16.png` (16×16 像素)
- `icon48.png` (48×48 像素)
- `icon128.png` (128×128 像素)

或使用任何在线工具生成，例如 [favicon-generator.org](https://www.favicon-generator.org/)

---

### 步骤 2: 加载到 Chrome 🚀

1. **打开扩展管理页面**
   - 地址栏输入：`chrome://extensions/`
   - 或菜单 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 右上角 **开发者模式** 开关打开

3. **加载本地扩展**
   - 点击 **加载未打包的扩展程序**
   - 选择本项目目录（`chrome-password-filler`）

4. **完成！**
   - 插件会出现在扩展列表中
   - 工具栏右侧会显示插件图标

## 使用演示 👨‍💻

### 添加账号

1. 访问任意网站的登录页
2. 点击插件图标 → **当前网站** 标签页
3. 输入用户名、密码、标签（可选）
4. 点击 **保存** 按钮

### 快速登录

1. 访问登录页面
2. 点击密码输入框
3. 选择弹出的账号列表中的一条
4. 自动填充，按 Enter 登录

### 查看所有账号

- 插件 Popup → **全部密码** 标签页
- 可以复制密码或删除记录

## 常见问题 ❓

### Q: 密码存储在哪里？
**A**: 本地浏览器 `chrome.storage.local`，存储在你的电脑上，不上传到云端。

### Q: 能否同步到其他设备？
**A**: 当前版本不支持。如需同步，可联系开发者或自行扩展功能。

### Q: 是否支持 Firefox/Edge？
**A**: 目前仅支持 Chrome。其他浏览器需要适配（Manifest V2 vs V3）。

### Q: 如何卸载？
**A**: Chrome 扩展页面 → 找到本插件 → 点击 **删除** → 确认。

### Q: 密码会被泄露吗？
**A**: 插件完全离线运行，密码不经过网络。但如你的电脑被黑客入侵，仍有风险。
高安全敏感账号（银行、支付）建议使用专业密码管理器如 Bitwarden。

## 故障排除 🔧

### 图标不显示
- 检查 `icons/` 目录是否存在 PNG 文件
- 刷新 Chrome 扩展页面 (F5)

### 登录页不显示建议
- 检查网站是否设置了内容安全策略 (CSP)
- 尝试刷新页面
- 打开 F12 开发者工具查看错误信息

### 填充后表单无反应
- 某些网站使用特殊表单库可能不兼容
- 可以手动复制密码使用

### 无法添加账号
- 确保输入了用户名和密码
- 检查 `chrome.storage` 权限是否被阻止
