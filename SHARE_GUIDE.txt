🚀 用户如何使用？
================================================================================

1. 解压 ZIP 文件
2. 打开 Chrome，访问：chrome://extensions/
3. 启用「开发者模式」（右上角开关）
4. 点击「加载未打包的扩展程序」
5. 选择刚才解压的文件夹
6. ✅ 完成！


💾 存储在哪里？
================================================================================

打包后的文件位置：
  → dist/chrome-password-filler-source.zip


🔑 重要：保管好私钥
================================================================================

⚠️ 这个文件很重要：
  → keys/private.pem

这是签名密钥，用于未来的版本更新。
  • 不要删除它
  • 不要分享给任何人
  • 建议备份到安全的地方
  • 上传 Git 时要排除它（.gitignore 已配置）


🎯 三种分发方案
================================================================================

【方案 1】最简单 - 直接分享 ZIP
  ✅ 用 ZIP 文件分享
  ✅ 用户自己解压和加载
  ✅ 最适合小团队

【方案 2】有签名 - 分享 CRX 文件
  ✅ 看起来更专业
  ✅ 用户可直接拖拽安装
  ✅ 需要生成步骤：npm run pack
  ✅ 文件位置：dist/chrome-password-filler.crx

【方案 3】最专业 - 上传到应用商店
  ✅ 发布到 Chrome Web Store
  ✅ 更新自动推送
  ✅ 需要 Google 账号和 $5 一次性费用


📝 常见问题
================================================================================

Q: 用户说看不到插件图标？
A: 让他检查有没有图标文件：
   icons/icon16.png, icon48.png, icon128.png

Q: 用户安装后没反应？
A: 让他在 chrome://extensions/ 点击刷新按钮

Q: 用户更新到新版本怎么办？
A: ZIP 方案：重新解压覆盖旧文件
   CRX 方案：用户重新下载新的 .crx 文件
